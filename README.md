This is the first blog entry of my project. Implementation was done before the blog began so many subsequent entries will be written post-factum. Main idea of the project is to implement a technique called "Fast N-body simulation with GPU on CUDA" from GPU Gems 3 book on Unity engine.(link) Core of the idea is multithreaded acceleration calculation done on the GPU which can be similarly implemented in Unity using a compute shader.
